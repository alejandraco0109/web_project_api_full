# Tripleten web_project_api_full

# Around The U.S. (Full Stack) - Proyecto Final
Este es el proyecto de cierre del programa de Desarrollo Web de Tripleten. Es una aplicación web completa que permite a los usuarios registrarse, iniciar sesión, compartir fotografías de lugares de interés, darles "like" y gestionar su perfil.

# Tecnologías Utilizadas
Frontend:
React (Vite)
React Router DOM (Navegación)
Context API (Gestión de estado del usuario)
CSS (Diseño responsivo basado en metodologías BEM)
Backend:
Node.js y Express
MongoDB y Mongoose (Base de datos NoSQL)
JWT (Autenticación y Autorización)
Bcryptjs (Encriptación de contraseñas)
Winston y Express-Winston (Logging de actividad)
Celebrate y Joi (Validación de datos)

# Estructura del Repositorio
/frontend: Código fuente de la aplicación React.
/backend: Servidor de Node.js con la API REST.
